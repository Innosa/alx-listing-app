# ALX Listing App
The project title: ALX Listing App.

## Description

ALX Listing App is a project aimed at creating an Airbnb-style listing page.  
The goal is to build a responsive and dynamic web application where users can view property listings with details such as images, prices, and descriptions.

## Project Structure
- **components/** – Contains reusable UI components such as cards, buttons, and layout elements.  
- **interfaces/** – Defines TypeScript interfaces used throughout the project for type safety.  
- **constants/** – Stores constant values such as configuration data and static variables.  
- **public/assets/** – Holds images, icons, and other static files used in the app.
